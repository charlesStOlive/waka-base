TalktoB publie des données

TalktoB vous permet de publier vos données SI sous de multiples formats.

  • Format Word pour les consultants et les commerciaux qui ont besoin d'ajouter leurs propres reflexions
  • Format email pour envoyer des informations ciblées
  • Format PDF pour des documents non modifiables et inviolables
  • Format Excel pour réutiliser les informations sur des outils extérieurs
  • des API pour nourrir vos autres applications.
